Sobre el Proyecto

Este sistema surge como un proyecto de Programación Web con el objetivo de brindar a los estudiantes una plataforma ágil para poner a prueba sus conocimientos antes de un examen real. Buscamos optimizar los tiempos de corrección mediante procesos automatizados.
